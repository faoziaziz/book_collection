** HotKey dalam emacs
| command     | deskripsi                |
| C - x C - f | untuk membuka file       |
| C - x C - s | untuk mengesave file     |
| C - x k     | untuk kill screen        |
| C - x C - c | menutup emacs            |
| C - x 2     | Split verticals          |
| C - x 3     | Split horizontal         |
| C - x o     | Pindah screen            |
| C - x 1     | Untuk memfokuskan screen |
